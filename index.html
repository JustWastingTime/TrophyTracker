How it works

  1. Download umadump.exe (v2.2.0). Run it while Umamusume (Steam) is open — it creates trophy_data.json.
  2. Upload that file here with Upload data.
  3. Pick the uma you're running a career with.
  4. Won races show a checkmark — click any card to toggle manually.
  5. Use Share link to save progress in the URL.
