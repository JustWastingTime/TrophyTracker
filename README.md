# Trophy Tracker

**Use the site:** [justwastingtime.github.io/TrophyTracker](https://justwastingtime.github.io/TrophyTracker/)

Track graded career races and work toward the **Completionist** title. Upload your `trophy_data.json`, pick an uma, and see what you still need on the calendar.

## Quick start

1. Download [umadump](https://github.com/Werseter/umadump/releases/tag/v2.2.0) and run it while Umamusume (Steam) is open to get `trophy_data.json`
2. Open the [tracker](https://justwastingtime.github.io/TrophyTracker/) and upload that file
3. Select your character and go

umadump is a third-party tool not endorsed by Cygames. Use it at your own risk.
